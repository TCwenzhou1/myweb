## 📈 技能使用评估指标

每次项目结束后评估：
1. **使用频率**: 哪些技能使用最多？
2. **效果评分**: 每个技能的帮助程度（1-5分）
3. **改进建议**: 如何更好地使用技能？
4. **新需求**: 需要哪些新技能？

## 🚀 技能调用最佳实践

### 立即调用时机：
1. **遇到重复性任务** → 立即搜索相关技能
2. **需要专业知识** → 使用对应领域技能
3. **安全敏感操作** → 先使用 Skill Vetter
4. **项目总结** → 使用 Self-Improving 学习

### 避免的情况：
1. 过度依赖单一技能
2. 在不合适的时机调用技能
3. 忽略安全审查（安装新技能时）
4. 不进行技能效果评估

## 🔍 技能发现策略

### 定期搜索：
1. 每月使用 **Find Skills** 搜索新技术
2. 关注热门技能类别更新
3. 根据项目需求定向搜索
4. 参考社区推荐技能

### 技能评估：
1. 新技能必须经过 **Skill Vetter** 审查
2. 在小项目中试用新技能
3. 记录使用体验和效果
4. 决定是否纳入常用技能库
