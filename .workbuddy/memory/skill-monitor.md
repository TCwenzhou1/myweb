# 🔍 实时技能监控器

## 🎯 监控目标
在项目执行过程中自动检测状态，及时提示使用相关技能，确保不遗漏任何有用的工具。

## 📈 技能使用效果跟踪

### 跟踪指标:
1. **调用次数**: 每个技能在项目中被调用的频率
2. **任务匹配度**: 技能是否解决了正确的问题
3. **时间节省**: 使用技能后节省的时间百分比
4. **质量提升**: 使用技能后任务完成质量的提升

### 评估公式:
```
技能效果评分 = (任务匹配度 * 0.4) + (时间节省 * 0.3) + (质量提升 * 0.3)
```

## 🎨 智能提示系统

### 提示规则:
1. **立即提示**: 检测到明显的技能匹配时立即提示
2. **建议性提示**: 检测到可能相关的技能时建议性提示
3. **教育性提示**: 介绍新技能的功能和用途
4. **上下文提示**: 根据当前项目阶段的技能需求提示

### 提示时机:
1. **项目开始时**: 介绍核心技能
2. **遇到新任务类型时**: 推荐相关技能
3. **任务复杂度增加时**: 建议更高级的技能
4. **需要协作时**: 推荐沟通和协作技能
5. **项目收尾时**: 推荐总结和改进技能
