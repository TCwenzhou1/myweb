## 📊 技能效果评估表

### 评估问题 (项目结束后)
1. **技能匹配度**: 技能是否解决了正确的问题？ (1-5分)
2. **时间节省**: 使用技能节省了多少时间？ (百分比)
3. **质量提升**: 任务完成质量是否有提升？ (1-5分)
4. **学习价值**: 从技能使用中学到了什么？

### 评分公式
```
综合评分 = (匹配度 × 0.4) + (时间节省 × 0.3) + (质量提升 × 0.3)
```

## 🚨 重要提醒

### 必须遵守的规则
1. **安全第一**: 安装新技能前必须使用 `skill-vetter`
2. **适时使用**: 不要在不合适的时机调用技能
3. **效果评估**: 每次使用后评估技能效果
4. **持续学习**: 定期搜索和尝试新技能

### 最佳实践
1. **项目开始时**调用核心技能
2. **遇到新任务类型**时搜索相关技能
3. **定期反思**技能使用效果
4. **分享经验**到知识库

### 避免的陷阱
1. 过度依赖少数技能
2. 忽略技能安全审查
3. 不进行技能效果评估
4. 不及时更新技能库
